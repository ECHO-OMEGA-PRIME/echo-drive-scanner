## Getting Started

### Prerequisites

- Node.js 20+ or Python 3.11+
- Git

### Installation

```bash
git clone https://github.com/ECHO-OMEGA-PRIME/intelligent-drive-scanner.git
cd intelligent-drive-scanner
pip install -r requirements.txt
```

### Configuration

Copy `.env.example` to `.env` and fill in your API keys and configuration values.

### Development

```bash
npm start
```
